请您在修改，增加，删除文章时遵守一下规范

1.CID和ID分别是角色和游戏的编号，请勿重复，请按照引导填写

2.图片路径为/img/game/ID.png(未实装)或/img/chara/CID.png(已实装)

3.随意填写文章数据可能会导致数据库错误，可能需要从数据库手动删除

4.在编写主题内容时请遵守一下规则

(1)内容的总结构分为目录，图片栏和正文

(2)目录部分应使用"ul"标签, class应设置为content-index.

(3)图片部分应使用图片展示框架, 使用div标签,class=imge,下设img元素和div标签(div标签的class="imgdescrb").

5.正文部分标签

下拉式内容栏:内容请使用div标签，class="dropdown-content-relative",开关按钮处请使用onclick="DisplayContent(id)"

具体内容请参看篝之雾枝
